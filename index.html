Course Projects

Quality Assurance for Simple Java Text Editor
Fall 2022

In this project for my Quality Assurance (CMU 17623) course, me and my groupmates extended the PHNotePad Editor to support the following additional functionalities: Syntax Highlighting, Bracket Matching, Find all, Find direction, Ignore Case. We then used the tools and techniques taught to over the semester to evaluate the quality of our project. The testing techniques we used include: Static Analysis, blackbox testing, whitebox testing, unit testing, regression testing, integration testing, end-to-end testing, and test doubles. The testing tools we used include: Jacoco, Pitest, Mockito, SpotBugs and CheckStyle. The image shows a screenshot of the find functionalities, which I contributed most.
